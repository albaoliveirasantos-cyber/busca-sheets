 BuscaOffline Sheets

Aplicativo PWA (Progressive Web App) para busca offline em grandes planilhas CSV (13.000+ linhas).

## Funcionalidades

- **100% Offline**: Funciona sem internet após o primeiro acesso.
- **Alta Performance**: Usa IndexedDB para suportar milhares de linhas sem travar o celular.
- **PWA**: Pode ser instalado na tela inicial do Android e iOS.
- **Busca Inteligente**: Filtra por 4 colunas simultaneamente.
- **IA Integrada**: Resumo inteligente dos resultados (requer chave API opcional).

## Como usar

1. Acesse o link do aplicativo.
2. Exporte sua planilha do Google Sheets como **.csv**.
3. Faça o upload no aplicativo.
4. Adicione à tela inicial do celular para acesso rápido.

## Desenvolvimento

Para rodar localmente:

```bash
npm install
npm run dev
```
